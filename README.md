# 🏠 Real Estate Auction Dashboard - Power BI

A **Power BI dashboard** built for **Real Estate Auction Analytics** to track **live bidding activity, premium trends, property status, and city-wise auction performance**.

## 📊 Dashboard Overview
This dashboard provides an interactive view of real estate auction data and helps analyze auction performance through key business metrics and visual insights.

## 🔍 Key Insights
- **55 Live Auctions**
- **39.86 Premium Trend %**
- **60.98 Avg Bid Premium %**
- **Active Bidders Today**
- **City-wise Auction Value Analysis**
- **Property Status Breakdown**
- **Locality-wise Bidding Insights**

## 📈 Features
- Track **live property auctions**
- Monitor **bid premium trends over time**
- Compare **city-wise auction values**
- Analyze **property status** such as Live, Sold, Closing Soon, Listed, and Closed
- View **locality-level bidding performance**
- Support **data-driven decision-making** in the real estate auction domain
